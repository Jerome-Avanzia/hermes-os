Version: 1.0

Status: Draft

Owner: AVANZIA

## AI-First

Design systems so AI agents can operate with clear context, bounded authority and measurable outcomes.

## Documentation-First

Document decisions, systems and workflows before implementation.

## Git-First

Use version-controlled documentation and changes as the operational record.

## Human-in-the-Loop

Keep people accountable for significant decisions, approvals and exceptions.

## Security-First

Apply least privilege, explicit access controls and safe defaults from the start.

## Simplicity Over Complexity

Prefer small, understandable systems that create durable leverage.
